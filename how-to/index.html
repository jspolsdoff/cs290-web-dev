Using HighCharts API in a Ruby on Rails Appliaction

Ever wonder how you to display data you have collected in a beautiful and easy to interpret format?

Well keep reading because I'm going to walk you through how to utilize the Highchart API to create an interactive graph similiar to what you would find on the Google Finance page.

Wow, so pretty!! We are going to make something like this.

This guide will be going over utilize the Highcharts API in a Ruby on Rails application. Ruby on what? Don’t worry I will go over what Ruby on Rails is in the next section. The good news for you is that the Highcharts API can be used in any web development project. The interaction with the API is really universal. The only difference is the road you take to get there. So even though I’m using Ruby on Rails it can be used in a Node.js or PHP application as well.

Alright so get ready to hop aboard because this train is about to leave. Now in order to keep this How-To focused on the HighCharts API I will be rather brief on explaining some parts of the Ruby on Rails framework. Don’t worry if you’re going to follow along though. I’ll be including links to supplemental sites that will provide you with documentation to help you out.

At the end of this how to you'll have a fully functional app that you can upload CSV files to and are displayed visually using the Highchart API

So if you’re ready to start include beautiful data visualization in your web development go ahead and click “Next” and let’s get started.
